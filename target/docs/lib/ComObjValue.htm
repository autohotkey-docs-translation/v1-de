ComObjValue() [v1.0.91+]

Ermittelt den Wert oder Pointer, der in einem COM-Wrapper-Objekt hinterlegt ist.

Wert := ComObjValue(ComObject)

Parameter

ComObject

Ein Wrapper-Objekt, das ein COM-Objekt oder einen typisierten Wert enthält.

Rückgabewert

Diese Funktion gibt einen vorzeichenfähigen 64-Bit-Integer zurück.

Allgemeine Bemerkungen

Diese Funktion ist nicht für den allgemeinen Gebrauch bestimmt.

Der Aufruf von ComObjValue ist äquivalent zu variant.llVal, wo ComObject als VARIANT-Struktur behandelt wird. Jedes Skript, das diese Funktion verwendet, muss darauf achten, welchen Wertetyp das Wrapper-Objekt enthält und wie es behandelt werden soll. Wenn zum Beispiel ein Interface-Pointer zurückgegeben wird, sollte Release nicht aufgerufen werden, aber AddRef könnte erforderlich sein, je nachdem, was das Skript mit dem Pointer macht.
