SoundBeep

Erzeugt einen Ton über den PC-Lautsprecher.

SoundBeep , Frequenz, Dauer

Parameter

Frequenz

Wenn leer oder weggelassen, wird standardmäßig 523 verwendet. Andernfalls geben Sie die Frequenz des Tons an, eine Zahl zwischen 37 und 32767. Kann ein Ausdruck sein.

Dauer

Wenn leer oder weggelassen, wird standardmäßig 150 verwendet. Andernfalls geben Sie die Dauer des Tons an (in Millisekunden). Kann ein Ausdruck sein.

Bemerkungen

Das Skript wartet, bis der Ton wiedergegeben wurde, bevor es fortfährt. Es ist auch möglich, dass das System während der Tonwiedergabe langsamer reagiert.

Wenn der Computer keine Soundkarte hat, wird ein Standardton über den PC-Lautsprecher wiedergegeben.

Um stattdessen vordefinierte Systemsounds abzuspielen, verwenden Sie den Sternchenmodus von SoundPlay.
