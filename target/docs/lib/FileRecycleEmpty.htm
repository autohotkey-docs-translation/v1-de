FileRecycleEmpty

Leert den Papierkorb.

FileRecycleEmpty , LaufwBuchst

Parameter

LaufwBuchst

Wenn leer oder weggelassen, wird der Papierkorb von allen Laufwerken geleert. Andernfalls geben Sie einen Laufwerksbuchstaben an, z.B. C:\.

Fehlerbehandlung

[v1.1.04+]: Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Weitere Informationen finden Sie unter Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Dieser Befehl funktioniert nur, wenn MS Internet Explorer 4 oder höher installiert ist.

Siehe auch

FileRecycle, FileDelete, FileCopy, FileMove

Beispiele

Leert den Papierkorb des C-Laufwerks.

FileRecycleEmpty, C:\
