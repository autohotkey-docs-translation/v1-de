EnvGet [v1.0.43.08+]

Ruft den Wert einer bestimmten Umgebungsvariable ab.

EnvGet, AusgabeVar, UmgebungVar

Parameter

AusgabeVar

Name der Ausgabevariable, in der der abgerufene Wert gespeichert werden soll.

UmgebungVar

Name der Umgebungsvariable, z.B. Path.

Bemerkungen

Wenn UmgebungVar einen leeren Wert hat oder nicht existiert, wird AusgabeVar leer gemacht.

Das Betriebssystem limitiert jede Umgebungsvariable auf 32 KB Text.

Siehe auch

EnvSet, #NoEnv, EnvUpdate, SetEnv, Run / RunWait

Beispiele

Ruft den Wert einer Umgebungsvariable ab und speichert ihn in AusgabeVar.

EnvGet, AusgabeVar, LogonServer

Ruft den Pfad des Ordners "Programme" ab und meldet ihn. Eine alternative Methode finden Sie unter RegRead-Beispiel #2.

EnvGet, AusgabeVar, % A_Is64bitOS ? "ProgramW6432" : "ProgramFiles"
MsgBox, Programmdateien sind in: %AusgabeVar%

Ermittelt und meldet den Pfad des Local-Verzeichnisses (AppData) des aktuellen Benutzers.

EnvGet, LocalAppData, LocalAppData
MsgBox, Die Anwendungsdaten von %A_UserName% sind in: %LocalAppData%
