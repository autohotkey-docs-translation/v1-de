Asc()

Gibt einen numerischen Wert zurück, der das erste Byte oder die erste UTF-16-Codeeinheit einer bestimmten Zeichenkette repräsentiert.

Nummer := Asc(Zkette)

Parameter

Zkette

Eine Zeichenkette, deren numerischer Wert abgerufen werden soll.

Rückgabewert

Diese Funktion gibt einen numerischen Wert im Bereich von 0 bis 255 (bei ANSI) oder von 0 bis 0xFFFF (bei Unicode) zurück. Einzelheiten finden Sie unter Unicode vs. ANSI. Wenn Zkette leer ist, gibt sie 0 zurück.

Bemerkungen

Diese Funktion ist äquivalent zu Transform, AusgabeVar, Asc.

Für Unicode-Ergänzungszeichen können Sie stattdessen Ord(Zkette) verwenden.

Siehe auch

Transform, Ord(), Chr()

Beispiele

Beide Mitteilungsfenster unten zeigen 116 an, weil nur das erste Zeichen berücksichtigt wird.

MsgBox, % Asc("t") 
MsgBox, % Asc("test")
