EnvDiv

Setzt eine Variable auf sich selbst geteilt durch den angegebenen Wert. Gleichbedeutend mit: Var /= Wert.

Veraltet: Dieser Befehl ist nicht für neue Skripte empfohlen. Verwenden Sie stattdessen Var := Var / Wert oder Var /= Wert.

EnvDiv, Var, Wert

Parameter

Var

Name der Variable, auf die die Operation angewendet werden soll.

Wert

Ein Integer, eine Floating-Point-Zahl oder ein Ausdruck.

Bemerkungen

Dieser Befehl entspricht der Kurzschreibweise: Var /= Wert.

Eine Division durch 0 führt zu einer Fehlermeldung beim Laden des Skripts (falls möglich); andernfalls macht sie die Variable leer.

Wenn Var oder Wert leer ist oder nicht mit einer Zahl beginnt, wird der Wert bei der Berechnung als 0 behandelt (außer bei interner Verwendung in einem Ausdruck wie Var := X /= Y).

Wenn Var oder Wert einen Dezimalpunkt enthält, ist das Endergebnis eine Floating-Point-Zahl im via SetFormat gesetzten Format. Andernfalls wird das Ergebnis gekürzt (z.B. 19 geteilt durch 10 wird 1 ergeben).

Beispiele

Setzt MeineZählung auf sich selbst geteilt durch 2.

EnvDiv, MeineZählung, 2

Dasselbe wie oben.

MeineZählung /= 2
