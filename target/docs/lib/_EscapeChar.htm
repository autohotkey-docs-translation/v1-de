#EscapeChar

Ändert das Escapezeichen des Skripts (z.B. umgekehrtes Häkchen zu umgekehrtem Schrägstrich).

Veraltet: Diese Direktive ist nicht für neue Skripte empfohlen. Verwenden Sie stattdessen das reguläre Escapezeichen (Akzent / umgekehrtes Häkchen).

#EscapeChar NeuesZeichen

Parameter

NeuesZeichen

Geben Sie ein einzelnes Zeichen an.

Bemerkungen

Mit dem Escapezeichen wird kenntlich gemacht, dass das nachfolgende Zeichen anders als normal interpretiert werden soll.

Wenn diese Direktive im Skript nicht angegeben ist, verhält sie sich so, als wäre sie auf Gravis / umgekehrtes Häkchen (`) gesetzt. Die entsprechende Taste befindet sich links oben auf einer englischen Tastatur oder rechts neben der ß-Taste auf einer deutschen Tastatur. Um ein alleinstehendes umgekehrtes Häkchen zu erzeugen, drücken Sie bei gedrückter Umschalttaste die Gravistaste und anschließend die Leertaste. Die Verwendung dieses Zeichens anstelle des umgekehrten Schrägstrichs verhindert, dass zwei umgekehrte Schrägstriche in Dateipfaden angegeben werden müssen.

Unter Escapesequenzen finden Sie eine vollständige Liste der Sequenzen und ihrer Ergebnisse (z.B. würde `n ein LF-Zeichen erzeugen).

#EscapeChar kann wie alle anderen Direktiven nicht bedingt ausgeführt werden.

Siehe auch

Die folgenden Direktiven existieren ebenfalls, werden aber nur selten verwendet:

#DerefChar #  ; Ersetzt das reguläre Dereferenzzeichen (Prozent).
#Delimiter /  ; Ersetzt das reguläre Trennzeichen (Komma).

Veraltet: Diese Direktiven sind nicht für neue Skripte empfohlen. Verwenden Sie stattdessen das reguläre Dereferenzzeichen (Prozent) und das reguläre Trennzeichen (Komma).

Beispiele

Ändert das reguläre Escapezeichen (`) in einen umgekehrten Schrägstrich.

#EscapeChar \
