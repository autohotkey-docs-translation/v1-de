IniDelete

Löscht einen Wert aus einer INI-Datei im Standardformat.

IniDelete, DateiName, Sektion , Schlüssel

Parameter

DateiName

Name der INI-Datei. Wenn kein absoluter Pfad angegeben ist, wird die Datei im A_WorkingDir-Verzeichnis vermutet.

Sektion

Name einer Sektion in der INI-Datei, erkennbar an den eckigen Klammern (die Klammern müssen in diesem Parameter weggelassen werden).

Key

Wenn weggelassen, wird die komplette Sektion gelöscht. Andernfalls geben Sie den Namen eines Schlüssels in der INI-Datei an.

Fehlerbehandlung

[v1.1.04+]: Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Weitere Informationen finden Sie unter Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Eine typische INI-Datei sieht wie folgt aus:

[SektionName]
Schlüssel=Wert

Siehe auch

IniRead, IniWrite, RegDelete

Beispiele

Löscht ein Schlüssel-Wert-Paar in Sektion2 aus einer INI-Datei im Standardformat.

IniDelete, C:\Temp\MeineDatei.ini, Sektion2, Schlüssel
