Tastatur- oder Mausmakros erstellen

Ein Makro ist eine Reihe von geskripteten Aktionen, die bei Bedarf "wiedergegeben" werden. Die häufigste Aktion eines Makros ist das Senden von simulierten Tastendrücken oder Mausklicks an ein oder mehrere Fenster. Solche Fenster reagieren auf Tastendrücke und Mausklicks, als hätte der Benutzer diese physisch gedrückt. Auf diese Weise lassen sich wiederkehrende Aufgaben schnell und zuverlässig automatisieren.

Eine der bequemsten Arten der Makrowiedergabe ist die Verwendung von Hotkeys oder Hotstrings. Zum Beispiel würde der folgende Hotkey eine leere E-Mail-Nachricht erstellen und für einen bestimmten Empfänger vorbereiten, die Sie vor dem Versenden noch anpassen können:

^!s::  ; Hotkey STRG+ALT+S.
if not WinExist("Posteingang - Microsoft Outlook")
    return  ; Falsches Outlook-Fenster geöffnet, also nichts tun.
WinActivate  ; Das oben gefundene Fenster aktivieren.
Send ^n  ; Eine neue/leere E-Mail mit STRG+N erstellen.
WinWaitActive Unbenannt - Nachricht
Send {Tab 2}Produktrückruf für ACME Rocket Skates  ; Betreffzeile setzen.
Send {Tab}Sehr geehrte Damen und Herren,{Enter 2}Wir haben soeben einen kleinen Fehler entdeckt ...  ; etc.
return  ; Diese Zeile dient zum Beenden des Hotkeys.

Hotkey-Makros wie das obige sind besonders nützlich für Aufgaben, die mehrmals am Tag durchgeführt werden. Makros, die weniger häufig verwendet werden, können dagegen in einem eigenen Skript gespeichert werden, das über eine Verknüpfung im Startmenü oder auf dem Desktop gestartet werden kann.
