ErrorLevel

ErrorLevel ist eine interne Variable, die gesetzt wird, um den Erfolg oder Misserfolg bestimmter Befehle zu kennzeichnen (nicht alle Befehle ändern den Wert von ErrorLevel). In der Regel steht 0 für Erfolg, jeder andere Wert für Misserfolg. Sie können ErrorLevel auch selbst setzen.

Beachten Sie, dass RunWait ErrorLevel auf den Exitcode des Programms setzt. Die meisten Programme liefern einen Exitcode von 0, wenn sie erfolgreich ausgeführt wurden.

Jeder Thread hat seinen eigenen ErrorLevel-Wert, d.h. wenn der aktuelle Thread von einem anderen Thread unterbrochen wird, hat der ursprüngliche Thread bei Wiederaufnahme immer noch seinen originalen ErrorLevel-Wert, nicht den des unterbrechenden Threads.

Hinweis: Da einige Befehle ErrorLevel auf einen Wert größer als 1 setzen, ist es besser, ErrorLevel auf einen Wert ungleich 0 zu überprüfen.

Beispiele

Wartet maximal 1 Sekunde, bis MeinFenster existiert. Wenn WinWait das Zeitlimit überschreitet, wird ErrorLevel auf 1 gesetzt, andernfalls auf 0.

WinWait, MeinFenster,, 1
if ErrorLevel   ; also nicht leer oder 0.
    MsgBox, Das Fenster existiert nicht.
else
    MsgBox, Das Fenster existiert.
