CLSID-Liste (Windows Class Identifier)

Bestimmte spezielle Verzeichnisse im Betriebssystem werden durch eindeutige Strings identifiziert. Einige dieser Strings können mit FileSelectFile, FileSelectFolder und Run verwendet werden. Zum Beispiel:

FileSelectFile, AusgabeVar,, ::{645ff040-5081-101b-9f08-00aa002f954e}  ; Wählt eine Datei im Papierkorb aus.
FileSelectFolder, AusgabeVar, ::{20d04fe0-3aea-1069-a2d8-08002b30309d}  ; Wählt einen Ordner im Arbeitsplatz/Computer aus.

Die "Ja"-Einträge in der letzten Spalte sind nicht verbindlich: der Run-Befehl unterstützt je nach Systemkonfiguration möglicherweise unterschiedliche CLSIDs. Um ein CLSID-Ordner mit Run zu öffnen, fügt die CLSID in den ersten Parameter ein. Zum Beispiel:

Run ::{20d04fe0-3aea-1069-a2d8-08002b30309d}  ; Öffnet den Arbeitsplatz/Computer.
Run ::{645ff040-5081-101b-9f08-00aa002f954e}  ; Öffnet den Papierkorb.
Run ::{450d8fba-ad25-11d0-98a8-0800361b1103}\Eigener Ordner  ; Öffnet einen Ordner in"Eigene Dokumente".
Run %A_MyDocuments%\Eigener Ordner  ; Wie oben, funktioniert aber in den meisten Systemen.
