Erstellung eines Tastatur- oder Maus-Makros

Ein Makro besteht aus mehreren programmierten Aktionen, die auf Anfrage ausgeführt werden. Die häufigste Tätigkeit eines Makros ist das Senden von künstlichen Tastatureingaben und Mausklicks an ein oder mehreren Fenstern. Solche Fenster reagieren auf Tastatureingaben und Mausklicks, als hättest du sie manuell durchgeführt. Sich wiederholende Aufgaben können dadurch schnell und zuverlässig automatisiert durchgeführt werden.

Makros können per Hand geschrieben werden. Allerdings könnte es einfacher sein, umfassende Makros mithilfe eines Makro-Rekorders zu erstellen, wie z. B. mit dem Rekorder-Script von Titan/polyethene. Dieses Script beobachtet, was du eingibst und worauf du klickst, und berücksichtigt, welches Fenster gerade aktiv ist. Es überträgt diese Aktionen in ein funktionierendes Makro, so dass sie später mit schnellerem Tempo wiedergegeben werden können.

Eine bequeme Art zum Wiedergeben eines Makros ist es, ihn an ein Hotkey oder Hotstring zu binden. Zum Beispiel würde der folgende Hotkey eine leere E-Mail-Nachricht erstellen und für einen bestimmten Empfänger vorbereiten, so dass du sie vor dem Senden anpassen kannst:

^!s::  ; Hotkey STRG+ALT+S.
IfWinNotExist Posteingang - Microsoft Outlook
    return  ; Das gewünschte Fenster von Outlook ist nicht offen, also nichts tun.
WinActivate  ; Aktiviert das zuletzt gefundene Fenster.
Send ^n  ; Erstellt eine neue/leere E-Mail mit STRG+N.
WinWaitActive Unbenannt - Nachricht
Send {Tab 2}Produktrückruf für ACME Rocket Skates  ; Setzt die Betreffszeile.
Send {Tab}Sehr geehrte Damen und Herren,{Enter 2}Wir haben vor Kurzem einen kleinen Fehler entdeckt ...  ; etc.
return  ; Diese Zeile dient zum Beenden des Hotkeys.

Hotkey-Makros wie oben sind besonders für Aufgaben nützlich, die du mehrmals am Tag durchführst. Dagegen können Makros, die seltener verwendet werden, in einem einzelnen Script aufbewahrt werden, das über eine Verknüpfung im Startmenü oder auf dem Desktop aufgerufen werden kann.

Bitte schau dir das Schnellstart-Tutorial an, wie du eigene Makros und Hotkeys erstellen kannst.
