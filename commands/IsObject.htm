IsObject() [v1.0.90+]

Gibt eine Nummer ungleich 0 zurück, wenn ein bestimmter Wert ein Objekt ist.

TrueOderFalse := IsObject(Objektwert)

Parameter

Objektwert

Ein Objekt, das in einer Variable gespeichert ist, von einer Funktion zurückgegeben wurde, in einem anderen Objekt gespeichert ist oder direkt angegeben ist.

Rückgabewert

Diese Funktion gibt 1 zurück, wenn Objektwert ein Objekt ist; ansonsten 0.

Siehe auch

Objekte

Beispiele

#1

Objekt := {Key: "Wert"}

if IsObject(Objekt)
    MsgBox, Das ist ein Objekt.
else
    MsgBox, Das ist kein Objekt.
