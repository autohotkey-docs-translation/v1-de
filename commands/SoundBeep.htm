SoundBeep

Erzeugt einen Ton vom PC-Lautsprecher.

SoundBeep [, Frequenz, Dauer]

Parameter

Frequenz

Die Frequenz des Sounds, das auch ein Ausdruck sein kann. Es sollte eine Nummer zwischen 37 und 32767 sein. Ist dieser Parameter nicht vorhanden, gilt standardmäßig eine Frequenz von 523.

Dauer

Die Abspieldauer des Sounds in Millisekunden, das auch ein Ausdruck sein kann. Ist dieser Parameter nicht vorhanden, gilt standardmäßig eine Dauer von 150.

Bemerkungen

Das Script wartet, bis der Sound wiedergegeben wurde, bevor es fortfährt. Darüber hinaus könnte die Reaktionsfähigkeit des Systems beim Erzeugen des Sounds reduziert sein.

Wenn der Computer keine Soundkarte hat, wird ein Standard-Piepton über den PC-Lautsprecher wiedergegeben.

Um die Standardsounds des Systems anstelle von Pieptönen zu erzeugen, siehe Sternchen-Modus von SoundPlay.

Siehe auch

SoundPlay

Beispiel

SoundBeep  ; Abspielen der Standard-Tonhöhe und -dauer.
SoundBeep, 750, 500  ; Abspielen einer höheren Tonhöhe für eine halbe Sekunde.
