SoundBeep

Erzeugt einen Ton aus dem PC-Lautsprecher.

SoundBeep , Frequenz, Dauer

Parameter

Frequenz

Die Frequenz des Sounds, das auch ein Ausdruck sein kann. Es sollte eine Nummer zwischen 37 und 32767 sein. Lässt man diesen Parameter weg, wird standardmäßig eine Frequenz von 523 verwendet.

Dauer

Die Abspieldauer des Sounds in Millisekunden, das auch ein Ausdruck sein kann. Lässt man diesen Parameter weg, wird standardmäßig eine Dauer von 150 verwendet.

Bemerkungen

Das Skript wartet, bis der Sound wiedergegeben wurde, bevor es fortfährt. Darüber hinaus könnte die Reaktionsfähigkeit des Systems beim Erzeugen des Sounds reduziert sein.

Wenn der Computer keine Soundkarte hat, wird ein Standard-Piepton über den PC-Lautsprecher wiedergegeben.

Um vordefinierte Systemklänge anstelle von Pieptönen zu erzeugen, verwenden Sie den Sternchen-Modus von SoundPlay.
