If Var [not] between Untergrenze and Obergrenze

Überprüft, ob der Inhalt einer Variable zwischen zwei Werten liegt (numerisch oder alphabetisch).

if Var between Untergrenze and Obergrenze
if Var not between Untergrenze and Obergrenze

Parameter

Var

Name der Variable, deren Inhalt überprüft werden soll.

Untergrenze

Dieser Parameter bestimmt, wie niedrig Var sein darf, um innerhalb des angegebenen Bereichs zu bleiben. Kann eine Zeichenkette, Nummer oder Variablenreferenz sein.

Obergrenze

Dieser Parameter bestimmt, wie hoch Var sein darf, um innerhalb des angegebenen Bereichs zu bleiben. Kann eine Zeichenkette, Nummer oder Variablenreferenz sein.

Bemerkungen

Sind alle drei Parameter rein numerisch, werden sie als Zahlen anstatt als Zeichenketten verglichen. Ansonsten werden sie alphabetisch als Zeichenketten verglichen (das heißt, dass die alphabetische Reihenfolge bestimmen wird, ob Var im angegebenen Bereich liegt). In diesem Fall kann StringCaseSense On genutzt werden, um den Vergleich Groß-/Kleinschreibung-sensitiv zu machen.

Vorsicht: Die Operatoren "between", "is", "in" und "contains" werden in Ausdrücken nicht unterstützt.

Siehe auch

IfEqual/Greater/Less, if Var in/contains Vergleichsliste, if Var is Typ, IfInString, StringCaseSense, EnvAdd, Blöcke, Else

Beispiele

#1

if var between 1 and 5
    MsgBox, %var% liegt zwischen 1 und 5.

if var not between 0.0 and 1.0
    MsgBox %var% liegt nicht zwischen 0.0 und 1.0.

if var between %Niedrig% and %Hoch%
    MsgBox %var% liegt zwischen %Niedrig% und %Hoch%.

if var between Blau and Rot
    MsgBox %var% liegt zwischen den Wörtern Blau und Rot (alphabetisch).

#2

Untergrenze := 1
Obergrenze := 10
InputBox, Benutzereingabe, Bitte eine Zahl zwischen %Untergrenze% und %Obergrenze% eingeben
if Benutzereingabe not between %Untergrenze% and %Obergrenze%
    MsgBox Ihre Eingabe liegt nicht im gültigen Bereich.
