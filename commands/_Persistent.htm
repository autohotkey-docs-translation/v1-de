#Persistent

Hält ein Script dauerhaft aufrecht (bis der Benutzer das Script schließt oder ein ExitApp vorkommt).

#Persistent

Kommt diese Direktive irgendwo im Script vor, wird das Script nach Durchführung des automatischen Ausführungsbereichs (oberster Bereich des Scripts) weiterlaufen. Das ist für ein Script nützlich, das Timer und/oder benutzerdefinierte Menüpunkte enthält, aber nicht Hotkeys, Hotstrings, OnMessage() oder Gui.

Wenn diese Direktive in einem vorhandenen Script eingefügt wird, sollten einige oder alle Vorkommen von Exit mit ExitApp ersetzt werden, weil Exit einen persistenten Script nicht beenden würde, sondern nur den aktuellen Thread.

Seit v1.0.16 macht diese Direktive ein Script zu einer Einzelinstanz. Um dies zu überschreiben oder das Verhalten einer Einzelinstanz zu ändern, siehe #SingleInstance.

Verwandte Befehle

#SingleInstance, SetTimer, Menu, Exit, ExitApp
