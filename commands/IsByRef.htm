IsByRef() [v1.1.01+]

Gibt eine Nummer ungleich 0 zurück, wenn ein bestimmter ByRef-Parameter eine Variable übergeben bekommen hat.

TrueOderFalse := IsByRef(ParameterVar)

Parameter

ParameterVar

Eine Referenz zu der Variable. Zum Beispiel: IsByRef(MeinParameter).

Rückgabewert

Diese Funktion gibt eine 1 zurück, falls ParameterVar ein ByRef-Parameter ist und der Aufrufer eine Variable übergeben hat; oder 0, falls ParameterVar ein anderer Variablentyp ist.

Siehe auch

ByRef-Parameter

Beispiele

Meldet 1 (true), weil Param ein ByRef-Parameter ist und eine Variable bereitgestellt bekommen hat.

MsgBox, % Funktion(MeineVar)

Funktion(ByRef Param)
{
    return IsByRef(Param)
}
