ObjAddRef() / ObjRelease() [AHK_L 53+]

Erhöht oder verringert die Referenzzählung eines Objekts.

ObjAddRef(Ptr)
ObjRelease(Ptr)

Parameter

Ptr Ein nicht verwalteter Pointer eines Objekts oder einer COM-Schnittstelle.

Rückgabewert

Diese Funktionen geben die neue Referenzzählung zurück. Dieser Wert sollte nur für das Debugging verwendet werden.

Verwandte Befehle

Obwohl in den folgenden Artikeln die Referenzzählung in Bezug auf COM diskutiert wird, decken sie einige wichtige Konzepte und Regeln auf, die generell auch für AutoHotkey-Objekte gelten: IUnknown::AddRef, IUnknown::Release, Regeln für die Referenzzählung.

Beispiele

Siehe ComObjConnect.

obj := Object()

; Die folgenden zwei Zeilen sind gleichwertig:
ptr1 := Object(obj)
ptr2 := ObjectToPointer(obj)

ObjectToPointer(obj) {
    If !IsObject(obj)
        Return ""
    ptr := &obj

    ObjAddRef(ptr)
    Return ptr
}

; Jeder Pointer, der über Object() oder ObjectToPointer() ermittelt wird, muss manuell freigegeben werden,
; damit das Objekt freigegeben und der verwendete Speicher wiedergewonnen werden kann.
ObjRelease(ptr2)
ObjRelease(ptr1)
