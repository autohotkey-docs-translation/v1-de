GroupClose

Schließt das aktive Fenster, das gerade mit GroupActivate oder GroupDeactivate aktiviert wurde, und aktiviert das nächste Fenster in der Gruppe. Es kann auch alle Fenster einer Gruppe schließen.

GroupClose, Gruppenname , A|R

Parameter

Gruppenname

Name der Gruppe, die ursprünglich mit GroupAdd definiert wurde.

A|R

Wenn der Buchstabe A angegeben ist, werden alle Mitglieder der Gruppe geschlossen. Hat den gleichen Effekt wie WinClose ahk_group Gruppenname.

Ansonsten: Nachdem der Befehl das aktive Fenster geschlossen hat, wird das nächste Fenster in der Reihe aktiviert. Dieser Parameter entscheidet, ob das älteste oder das neueste Fenster aktiviert werden soll. Fehlt dieser Parameter, wird immer das älteste Fenster aktiviert. Wenn der Buchstabe R angegeben ist, wird das neueste (zuletzt aktive) Fenster aktiviert, aber nur, wenn beim Ausführen des Befehls kein Fenster aus der Gruppe aktiv ist. "R" ist zum Beispiel nützlich, wenn Sie vorübergehend zu einer anderen Aufgabe wechseln müssen. Wenn Sie mit GroupActivate, GroupDeactivate oder GroupClose zur Gruppe zurückkehren, wird das Fenster, mit dem Sie zuletzt gearbeitet haben, aktiviert und nicht das älteste Fenster.

Bemerkungen

Wenn der A|R-Parameter nicht der Buchstabe A ist, wird das Verhalten dieses Befehls dadurch bestimmt, ob die vorherige Aktion auf Gruppenname GroupActivate oder GroupDeactivate war. Wenn es GroupDeactivate war, wird dieser Befehl das aktive Fenster nur schließen, wenn das Fenster kein Mitglied der Gruppe ist (ansonsten passiert nichts). Wenn es GroupActivate oder nichts war, wird dieser Befehl das aktive Fenster nur schließen, wenn das Fenster ein Mitglied der Gruppe ist (ansonsten passiert nichts). Dieses Verhalten erlaubt es, GroupClose einem Hotkey zuzuweisen, als Ergänzung zum Gruppenname's GroupActivate- oder GroupDeactivate-Hotkey.
