IfMsgBox

Überprüft, welche MsgBox-Schaltfläche der Benutzer zuletzt gedrückt hat.

IfMsgBox, SchaltflName

Parameter

SchaltflName

Eine der folgenden Zeichenketten, die die zuletzt gedrückte MsgBox-Schaltfläche repräsentiert:

  • Yes
  • No
  • OK
  • Cancel
  • Abort
  • Ignore
  • Retry
  • Continue [v1.0.44.08+]
  • TryAgain [v1.0.44.08+]
  • Timeout (wenn MsgBox die Zeit überschreitet)

Siehe auch

MsgBox

Beispiele

#1: Zeigt ein Ja-Nein-Mitteilungsfenster an, das die Ausführung nach 5 Sekunden automatisch beendet. Der Benutzer kann die Schaltfläche "Nein" drücken, um den Stopp sofort einzuleiten.

MsgBox, 4, , Möchten Sie fortfahren?, 5  ; 5-sekündliches Zeitlimit.
IfMsgBox, No
    Return  ; Benutzer hat die Schaltfläche "Nein" gedrückt.
IfMsgBox, Timeout
    Return ; "Nein" vermuten, wenn es die Zeit überschritten hat.
; Ansonsten fortsetzen:
; ...
