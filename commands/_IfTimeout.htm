#IfTimeout [AHK_L 8+]

Bestimmt die maximale Zeit, die zur Auswertung eines einzelnen #If-Ausdrucks aufgebracht werden soll.

#IfTimeout Timeout

Parameter

Timeout Timeout-Wert in Millisekunden, der global gelten soll.

Bemerkungen

Das Timeout wurde implementiert, um lang laufende Ausdrücke daran zu hindern, den Tastatureingabeprozess hinauszuzögern. Sobald der Timeout-Wert überschritten ist, wird der Ausdruck weiter ausgewertet. Der Tastatur-Hook interpretiert den Ausdruck jedoch so, als wäre er bereits als falsch gewertet wurden.

Ist diese Direktive im Script nicht angegeben, gilt standardmäßig 1000.

Beachte, dass das System einen eigenen Timeout implementiert hat. Sobald das System-Timeout überschritten ist, wird möglicherweise sowohl der Hotkey als auch die ursprüngliche Funktion der Taste aktiviert. Das System-Timeout ist im DWORD-Wert LowLevelHooksTimeout definiert, der wiederum im folgenden Registry-Key gefunden werden kann:

HKEY_CURRENT_USER\Control Panel\Desktop

Verwandte Befehle

#If

Beispiel

#IfTimeout 10 ; Setzt das Timeout auf 10 ms.
