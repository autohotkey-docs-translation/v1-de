If (Ausdruck)

Ein oder mehrere Befehle, die ausgeführt werden, wenn ein Ausdruck als WAHR ausgewertet wird.

If (Ausdruck)

Bemerkungen

Eine If-Anweisung, die ein Ausdruck enthält, unterscheidet sich von einem traditionellen If wie If Fundfarbe <> Blau, sobald das Zeichen nach dem Wort "If" eine öffnende Klammer ist. Obwohl das in der Regel erreicht wird, wenn der komplette Ausdruck in Klammern gesetzt wird, ist das beispielsweise auch mit If (x > 0) and (y > 0) möglich. Darüber hinaus kann die öffnende Klammer komplett weggelassen werden, wenn das erste Element nach dem Wort "If" ein Funktionsaufruf oder ein Operator wie "not" oder "!" ist.

Falls der Ausdruck der If-Anweisung als wahr gewertet wird (das heißt, dass das Ergebnis weder leer noch eine 0 sein darf), wird die Zeile oder der Block darunter ausgeführt. Ansonsten wird die Zeile oder der Block beim nachfolgenden ELSE ausgeführt, falls vorhanden.

Sobald IF oder ELSE mehrere Zeilen enthält, müssen diese mit Klammern umschlossen werden. Ist allerdings nur eine Zeile vorhanden, können die Klammern weggelassen werden. Entsprechende Beispiele können im untersten Abschnitt gefunden werden.

Optional kann der "One True Brace Style" bei If-Anweisungen verwendet werden, die Ausdrücke sind (aber nicht traditionelle If-Anweisungen). Zum Beispiel:

If (x < y) {
    ...
}
If WinExist("Unbenannt - Editor") {
    WinActivate
}
If IsDone {
    ...
} Else {
    ...
}

Im Gegensatz zu Else-Anweisungen, bei denen auf der rechten Seite alle möglichen Anweisungsarten möglich sind, unterstützen If-Anweisungen nur einen "{" danach.

Auf ähnliche Weise kann mit der Anweisung If Variable [not] between Untergrenze and Obergrenze überprüft werden, ob eine Variable zwischen zwei Werten liegt, sowie mit If Variable [not] in Wert1,Wert2 überprüft werden, ob der Variableninhalt in einer Werteliste vorkommt.

Verwandte Befehle

Ausdrücke, Zuweisungsausdruck (:=), If Variable in/contains Übereinstimmungsliste, If Variable between, IfInString, Blocks, Else, While-Loop

Beispiel

If (A_Index > 100 or Fertig)
    Return

If (A_TickCount - Startzeit > 2*Maximalzeit + 100)
{
    MsgBox Zu viel Zeit verstrichen.
    ExitApp
}

If (Farbe = "Blau" or Farbe = "Weiß")
{
    MsgBox Die Farbe ist einer der erlaubten Werte.
    ExitApp
}
Else If (Farbe= "Silber")
{
    MsgBox Silber ist keine erlaubte Farbe.
    Return
}
Else
{
    MsgBox Diese Farbe wurde nicht erkannt.
    ExitApp
}
