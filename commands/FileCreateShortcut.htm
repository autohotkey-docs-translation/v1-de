FileCreateShortcut

Erstellt eine Verknüpfung (.lnk).

FileCreateShortcut, Ziel, Verknüpfung [, Arbeitsverz, Parameter, Beschreibung, Icondatei, Tastenkürzel, Iconnummer, Ausführungsstatus]

Parameter

Ziel

Name der zu verknüpfenden Datei, der einen absoluten Pfad beinhalten sollte, falls die Datei noch nicht im System integriert ist (z. B. Notepad.exe). Die Datei muss beim Erstellen der Verknüpfung nicht vorhanden sein. Das heißt, dass Verknüpfungen mit ungültigem Ziel erstellt werden können.

Verknüpfung

Name der Verknüpfung, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Die Erweiterung .lnk muss mit angegeben werden. Ist die Datei bereits vorhanden, wird sie überschrieben.

Arbeitsverz

Verzeichnis, das das Arbeitsverzeichnis von Ziel sein soll, sobald die Verknüpfung gestartet wird. Ist dieser Parameter leer oder nicht vorhanden, wird die Verknüpfung ein leeres "Ausführen in"-Feld haben und der Verknüpfung ein Standard-Arbeitsverzeichnis vom System bereitgestellt, sobald sie ausgeführt wird.

Parameter

Parameter, die an Ziel übergeben werden sollen, sobald die Verknüpfung gestartet wird. Trenne jeden Parameter mit Leerzeichen. Enthält ein Parameter selbst Leerzeichen, umschließe ihn mit Anführungszeichen.

Beschreibung

Kommentare, die die Verknüpfung beschreiben (wird beispielsweise zum Anzeigen des ToolTips verwendet).

Icondatei

Der vollständige Pfad und Name des Icons, das für Verknüpfung angezeigt werden soll. Muss entweder eine ICO-Datei oder das erste Icon einer EXE- oder DLL-Datei sein.

Tastenkürzel

Buchstabe, Zahl oder Name einer einzelnen Taste aus der Tastenliste (Maustasten und andere untypische Tasten werden möglicherweise nicht unterstützt). Modifikatorzeichen dürfen nicht angegeben werden. Zurzeit werden alle Tastenkürzel mit STRG+ALT kombiniert. Wenn zum Beispiel der Buchstabe B für diesen Parameter verwendet wird, gilt die Tastenkombination STRG+ALT+B.

Iconnummer

Um anstelle des ersten Icons ein beliebiges Icon in Icondatei zu verwenden, kann hier die Nummer angegeben werden (kann ein Ausdruck sein). Zum Beispiel ist 2 das zweite Icon.

Ausführungsstatus

Um Ziel minimiert oder maximiert zu starten, kann eine der folgenden Zahlen angegeben werden:

1 - Normal (Standard)
3 - Maximiert
7 - Minimiert

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Ein Pfad muss nicht in Ziel angegeben sein, wenn sich die Zieldatei in einem der Ordner befindet, die in der PATH-Umgebungsvariable aufgelistet sind.

Das Tastenkürzel einer neu erstellten Verknüpfung funktioniert nur, wenn die Verknüpfung auf dem Desktop oder irgendwo im Startmenü vorkommt. Ist das gewünschte Tastenkürzel bereits in Benutzung, hat deine neue Verknüpfung Vorrang.

Im folgenden Beispiel wird gezeigt, wie auf alternative Weise eine URL-Verknüpfung erstellt werden kann. Passe die ersten beiden Parameter deinen Bedürfnissen an:
IniWrite, http://www.google.com, C:\Meine Verknüpfung.url, InternetShortcut, URL

Je nach Bedarf kann das folgende Beispiel hinzugefügt werden, um den obigen Beispiel ein Icon zu zuweisen:

IniWrite, <Icondatei>, C:\Meine Verknüpfung.url, InternetShortcut, IconFile
IniWrite, 0, C:\Meine Verknüpfung.url, InternetShortcut, IconIndex

Ersetze 0 mit dem Index des gewünschten Icons (0 ist das erste Icon) und ersetze <Icondatei> mit einer URL-, EXE-, DLL- oder ICO-Datei. Beispiele: C:\Icons.dll, C:\App.exe, http://www.somedomain.com/ShortcutIcon.ico

Das Betriebssystem wird eine URL-Datei, die auf diese Weise erstellt wurde, als Verknüpfung behandeln, auch wenn es eine Klartextdatei ist, anstatt eine LNK-Datei.
