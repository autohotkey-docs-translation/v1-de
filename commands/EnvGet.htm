EnvGet [v1.0.43.08+]

Ermittelt eine Umgebungsvariable.

EnvGet, AusgabeVar, UmgebungsVar

Parameter

AusgabeVar

Name der Variable, in der der String gespeichert werden soll.

UmgebungsVar

Der Name der Umgebungsvariable, die abgerufen werden soll. Zum Beispiel: EnvGet, AusgabeVar, Path.

Bemerkungen

Ist die angegebene Umgebungsvariable leer oder nicht vorhanden, wird AusgabeVar leer gemacht.

Das Betriebssystem limitiert jede Umgebungsvariable auf 32 KB.
